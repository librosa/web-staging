librosa.display.ChromaFJSFormatter#

class librosa.display.ChromaFJSFormatter(*, intervals, unison='C', unicode=True, bins_per_octave=None)[source]#

A formatter for chroma axes with functional just notation

Parameters:
intervalsstr or array of float in [1, 2)

The interval specification for the chroma axis. See core.interval_frequencies for supported values.

unisonstr

The unison (tonic) note name.

unicodebool

If True, use unicode symbols for accidentals.

If False, use ASCII symbols for accidentals.

bins_per_octaveint or None

The number of bins per octave. If None, inferred from intervals.
